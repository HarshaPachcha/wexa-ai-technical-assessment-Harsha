StockFlow MVP – Inventory Management Dashboard

StockFlow MVP is a simple SaaS-style inventory management system built using Flask, SQLAlchemy, and Chart.js.
It allows organizations to manage products, track inventory, record sales, and visualize stock analytics through charts.

---

Features

- User Signup and Login
- Organization-based inventory management
- Add products with SKU and quantity
- Sell products (automatically reduces stock)
- Delete products
- Low stock detection
- Sales history tracking
- Dashboard analytics with charts
- Inventory distribution visualization
- Stock health monitoring

Installation

Clone the repository

git clone https://github.com/Harshapachcha/wexa-ai-technical-assessment-Harsha.git

Navigate to project folder

cd wexa-ai-technical-assessment-Harsha

Install dependencies

pip install -r requirements.txt

Run the application

python app.py

Open browser

http://127.0.0.1:5000

---

Usage

1. Create an account using the signup page
2. Login using your credentials
3. Add products to inventory
4. Sell products and update stock
5. Monitor inventory analytics from dashboard charts
6. Track low stock alerts

---

Charts Available

- Inventory Distribution (Bar Chart)
- Stock Health (Pie Chart)
- Sales Analytics (Bar Chart)
